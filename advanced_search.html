Improving Search

Search engine พื้นฐานใช้เพียงการให้คะแนนแต่ละ term ที่อยู่ใน query และ document การพัฒนา search engine ต่อไปต้องอาศัยการประเมินผล ที่ถูกวิธี และการใช้ semantics model ในการเข้าถึงความหมายและความต้องการทางด้านข้อมูลของผู้ใช้

การประเมินประสิทธิภาพระบบการ Search

เราสามารถประเมินประสิทธิภาพของระบบด้วย Click data ที่ดึงออกมาจาก search log แต่ที่ดีกว่านั้นคือเราสามารถประเมินความสามารถในการตอบสนองต่อ information need ของผู้ใช้โดยการใช้ A/B testing

Semantic Search
