ระบบการ Search ขั้นพื้นฐาน

Search engine ทุกประเภทบนโลกตั้งอยู่บน data structure คล้ายๆ กัน และมีการจัดระเบียบข้อมูลที่ทำให้เครื่อง search ได้เร็ว และไม่ต้องเปลืองหน่วยความจำของเครื่องคอมพิวเตอร์ที่ใช้ในการตอบสนอง search query
