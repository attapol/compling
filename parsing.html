Syntax คือ การศึกษาโครงสร้างของประโยคและกฏไวยากรณ์ นักทำ NLP ควรทำความเข้าใจทฤษฎี Syntax คร่าวๆ เพื่อที่จะสามารถใช้ประโยชน์จาก Dependency parse ได้เต็มที่ และสามารถพัฒนา Dependency parser ให้มีประสิทธิภาพสูงขึ้น
