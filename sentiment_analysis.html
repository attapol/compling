Sentiment Analysis

การวิเคราะห์อารมณ์ความรู้สึกที่แสดงออกผ่านทางภาษา เทคโนโลยีนี้เป็นที่นิยมมากเนื่องจากข้อมูลทางด้านความคิดเห็นนั้น สามารถแปลงค่าไปเป็น applications และความรู้เชิงลึก (insights) เพื่อประกอบและชี้นำการตัดสินใจ

เทคโนโลยีนี้อาศัย machine learning ในการเรียนรู้จากข้อมูลประกอบกับทฤษฎีทางด้านจิตวิทยา และภาษาศาสตร์การวิเคราะห์อารมณ์
